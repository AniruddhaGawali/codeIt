C++

C++ is a powerful and versatile programming language known for its ability to handle complex tasks, combining features of both high-level and low-level programming with a syntax that resembles the English language.

C

C is a popular procedural programming language known for its performance and efficiency. It's widely used in system programming and developing low-level applications.

Dart

Dart is a modern programming language developed by Google, primarily used for building web and mobile applications with the Flutter framework.

Go

Go, also known as Golang, is a statically typed, compiled language developed by Google. It's known for its simplicity and efficiency and is commonly used in backend web development and system programming.

Java

Java is a popular and versatile programming language known for its "write once, run anywhere" capability. It's widely used in various domains, including web development, mobile app development, and enterprise software.

JavaScript

JavaScript is a widely-used scripting language for web development. It allows you to add dynamic behavior to websites and is an essential part of front-end web development.

Jupyter Notebook

Jupyter Notebook is an open-source web application that allows you to create and share documents containing live code, equations, visualizations, and narrative text. It's commonly used in data science and research.

Kotlin

Kotlin is a modern, statically-typed programming language that runs on the Java Virtual Machine (JVM). It's known for its conciseness and safety features and is commonly used in Android app development.

Object-Oriented Programming (OOP) Concepts

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes for organizing code. It focuses on encapsulation, inheritance, and polymorphism, among other principles, to create reusable and maintainable code.

Python

Python is a versatile and easy-to-learn programming language. It's widely used in web development, data science, machine learning, and automation due to its simplicity and extensive libraries.

Ruby

Ruby is a dynamic, object-oriented programming language known for its elegant syntax and developer-friendly features. It's commonly used for web development, particularly with the Ruby on Rails framework.

Rust

Rust is a systems programming language known for its emphasis on safety, performance, and concurrency. It's used for tasks where low-level control over hardware is required while still ensuring memory safety.

Swift

Swift is a programming language developed by Apple for building iOS, macOS, watchOS, and tvOS applications. It's known for its speed and developer-friendly syntax.

Web Development

Web development involves creating websites and web applications. It includes various technologies such as HTML, CSS, JavaScript, and server-side scripting languages like PHP, Ruby, and Python.
